    torch
    torchvision
    tensorboardX
    lion_pytorch

```
python train_val.py --dataset [cifar10/cifar100/imagenet] --Algorithm [SGDM/ASIGNSGD] --lr [learning_rate] --weight_decay [weight decay] --theta [theta] --zeta [zeta] 
```

